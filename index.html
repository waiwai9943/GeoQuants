About Corn Alpha

This project provides agricultural insights by correlating satellite imagery with market data for the US corn belt. It also includes a time-series forecasting feature using a SARIMA model to predict future vegetation trends.

Technical Details

  • Satellite Data: The analysis utilizes imagery from the Sentinel-2 constellation, which provides high-resolution (10m) multispectral data.
  • Data Aggregation: To ensure a clear and representative view of the selected area, cloudy images are filtered out, and the clearest observations over a 5-day window are aggregated using a median calculation. This reduces noise and provides a robust time-series dataset.
  • Key Indices: Both NDVI (Normalized Difference Vegetation Index) for overall plant health and NDRE (Normalized Difference Red Edge Index) for chlorophyll content are calculated and available for analysis.
